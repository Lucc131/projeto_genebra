# Projeto Genebra

## Sobre o Projeto

O Projeto Genebra é uma plataforma web desenvolvida com foco em acessibilidade, organização e apresentação de conteúdos cristãos. O objetivo do projeto é oferecer uma experiência intuitiva e agradável para usuários que desejam acessar informações, estudos, recursos e conteúdos relacionados ao universo bíblico e cristão.

O projeto foi construído utilizando tecnologias web fundamentais, priorizando simplicidade, desempenho, responsividade e compatibilidade com diferentes dispositivos.

## Objetivos

* Disponibilizar conteúdos de forma clara e organizada.
* Proporcionar uma experiência consistente em dispositivos móveis e desktops.
* Oferecer suporte aos modos claro e escuro para melhor conforto visual.
* Manter uma interface moderna, limpa e de fácil navegação.
* Garantir boa performance e acessibilidade para diferentes perfis de usuários.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## Funcionalidades

* Interface responsiva para diferentes tamanhos de tela.
* Alternância entre modo claro e modo escuro.
* Navegação intuitiva e organizada.
* Estrutura otimizada para leitura de conteúdos.
* Layout adaptado para dispositivos móveis.
* Componentes visuais desenvolvidos com foco em usabilidade.
